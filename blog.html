Question-1: What are the different ways to select an element in the DOM?
You can use getElementById, getElementsByClassName, querySelector, and querySelectorAll to select elements.
Question-2: What is the difference between innerHTML, innerText, and textContent?
innerHTML returns the HTML inside an element, innerText returns only visible text, and textContent returns all text, including hidden text.
Question-3: What is event delegation in the DOM?
Event delegation allows handling events at a parent level instead of individual elements using event bubbling.
Question-4: What is event bubbling in the DOM?
Event bubbling is when an event triggered on a child element propagates up to its parent elements.
Question-5: How do you create, add, and remove elements using JavaScript?
Use document.createElement() to create, appendChild() to add, and removeChild() to remove elements.
